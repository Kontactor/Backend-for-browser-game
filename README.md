### Проект: Серверная часть браузерной игры
#### Цель:
Реализация высокопроизводительного игрового сервера с поддержкой многопользовательского режима и системой рейтингов игроков.

#### Использованные технологии и библиотеки:
Boost Libraries: Asio для асинхронного ввода-вывода, Beast для высокоуровневого HTTP-программирования, JSON для обработки запросов, Log для отслеживания ошибок и диагностики, Serialization для сохранения игровых состояний.
Catch2: Фреймворк для модульного тестирования, обеспечивающего высокое качество и стабильность серверной части.
PostgreSQL + libpqxx: Создание эффективной схемы базы данных для быстрого доступа к данным рейтинга игроков.
Docker: Контейнеризация приложения для упрощения развёртывания и управления зависимостями.
Основные достижения:

Обеспечение надёжности сети и минимальной задержки соединения благодаря эффективному управлению соединениями и потоками.
Организация эффективного взаимодействия клиент-сервер посредством REST API и WebSocket-каналов связи.
Высокая производительность системы рейтинга и обработка большого количества одновременных пользователей.
Использование механизма сериализации для оптимизации хранения игровых сессий и состояния игроков.
Тестирование всех основных компонентов серверной части с помощью автоматизированных тестов, написанных на фреймворке Catch2.

#### Опыт:
Проектирование и разработка веб-сервисов с применением C++ и Boost.
Практический опыт работы с базами данных PostgreSQL и SQL-запросами.
Навык упаковки приложений в контейнеры Docker и разворачивания инфраструктуры.
Владение методологией разработки через тестирование (TDD).
